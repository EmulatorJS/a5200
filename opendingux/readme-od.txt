Race for OD V1.0
--------------------------------------------------------------------------------
This is a Atari 5200 emulator for the GCW-ZERO.
To use this emulator, you must use compatibles rom with a52/bin format. 
You also need the 5200.bin rom bios to use it. A quick search on atariage website 
will help you about that ;-)
Do not ask me about such files, I don't have them. A search with Google will certainly 
help you.

How to use ProSystem :
--------------------------------------------------------------------------------
Put a5200.opk to "apps" folder of inner flash-memory of your handheld.
Put bios named 5200.rom to "/boot/local/home" directory
Put your .A52 & .BIN games where you want.

Controls :
 * Direction pad & analog pad if active : A5200 pad
 * A is Fire button 1
 * B is Shift button
 * START is the same that the START button
 * SELECT is the same that the PAUSE button
 * X is button #*
 * Y is button ##
 * R is button #0
 * L is button #1
 
You can also click on START + SELECT to enter the menu.
